# NanoChiral-ML
**Computational Analysis of 1-D Quantum Confinement in Carbon Nanotubes**

## 🔬 Overview
This project investigates the relationship between structural chirality $(n, m)$ and atomic displacement in Single-Walled Carbon Nanotubes (SWCNTs). Using a Random Forest Regressor, the model achieves a **0.9974 R² score** in predicting atomic coordinates under 1-D confinement.

## 🛠️ Features
- **Predictive Modeling:** High-precision characterization of atomic shifts.
- **Interactive Dashboard:** Real-time parameter adjustment using Streamlit.
- **3D Visualization:** Dynamic structural representation via Plotly.

## 🚀 Installation & Usage
1. Clone the repo: `git clone https://github.com/your-username/NanoChiral-ML.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run main.py`
